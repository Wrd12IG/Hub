SEZIONE 6: RISOLUZIONE PROBLEMI
================================================================================

PROBLEMA: "Node non riconosciuto come comando"
SOLUZIONE: 
  - Installa Node.js da https://nodejs.org/
  - Riavvia il PC dopo l'installazione

PROBLEMA: L'app non si avvia / porta giÃ  in uso
SOLUZIONE:
  - Apri Task Manager (Ctrl+Shift+Esc)
  - Cerca processi "node.exe" e terminali
  - Prova a riavviare

PROBLEMA: La PWA non si installa
SOLUZIONE:
  - Usa "Installa_PWA_Chrome.bat" per configurare Chrome
  - Assicurati di riavviare Chrome dopo la configurazione
  - Usa Chrome o Edge (Firefox non supporta pienamente le PWA)

PROBLEMA: "EADDRINUSE" - porta giÃ  in uso
SOLUZIONE:
  - Un'altra istanza del server Ã¨ in esecuzione
  - Chiudi tutte le finestre cmd relative all'Hub
  - Oppure usa Task Manager per terminare node.exe
